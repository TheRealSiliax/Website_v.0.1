### Agent-AI-Team Framework

Grundstruktur für Rollen, Prozesse, SOPs, Templates und Tool-Registry. Inhalte aus `docs/intake/` werden hier konsolidiert.

Ordner:
- `roles.yaml`: zentrale Rollenbeschreibung
- `agents/`: rollenbezogene Arbeitsanweisungen
- `processes/`: Phasen/Workflows
- `sops/`: konkrete Schritt-für-Schritt-Anleitungen
- `templates/`: Vorlagen für Protokolle, Tickets, ADRs, Experimente
- `tool_registry/`: Tools und Integrationen
